# Routes Naming Convention

route name: `parent.child`
route path: `/parent/child`
route view: `views/Parent/Child.vue` (parent: `views/Parent/index.vue`)

# Routes

  * 欢迎页 `Welcome`
    * 隐私权、TOS 简要版 `TermsPreview`
  * 登录 `Login`
  * 快速设置 `QuickSettings`
  * 主页 `Home`
  * 工具 `Apps`
    * 课表 `Schedule`
    * 一卡通 `Card`
    * 阳光长跑 `SunRun`
      * 排行榜 `Rank`
      * 个人详情 `Details`
      * 帮助 `Help`
    * 成绩 `Scores`
    * 考试 `Exam`
    * 图书馆 `Library`
      * 图书搜索 `Search`
      * 图书详情 `Details`
    * 电费 `Electricity`
    * 校历 `Calendar`
